Entity

Defined in src\core\entity.js:40

Adds new component to an entity. Component is either created from scratch or reused from pool. In later case, component patterns reset method is called (if present).
Component patterns initialize method is called with additional arguments passed to add method. Addition does not happen imediately, but is postponed to nearest update cycle.

Parameters:

  • name String multiple

    name of component to add. Addidtional parameters are applied to component patterns initialize method.

//code like this often can be seen in entity pattern create method
