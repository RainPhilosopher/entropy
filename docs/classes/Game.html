Method used to register event listener.

Parameters:

  • event String

    event name

  • fn Function

    event listener

  • binding Object

    context used as this when calling listener

  • once Boolean

    if set to true, listener will be called once, then will be unregistered

Example:

emitter.on('event', function () {
    console.log(this.foo); //bar
}, {foo: 'bar'});
